Съдръжание на архива:
	- папка node_modules включва библиотеки от node.js(набор от функции, които съм включил в моето приложение).
	- папка html, съдържаща .html файловете на приложението.
	- папка public с:
		- подпапка css, съдържаща .css файловете за html страниците
		- подпапка img, съдържаща изображението bg.jpg на началната страница на приложението.
	- папка sql, включваща файл colormusic.sql на базата данни 
	- папка config, включваща файл db.json, който вклюва настройки за създаване на връзка между сървъра и базата данни. 
	- package.json и package-lock.json са файлове, който са нужни на node package manager да изтегли dependancies 
	  на проекта.
	- файл server.js преставлява файл, който реализира сървърната логика.
	- пълна документация на проекта

Инструкции:

Проектът е пускан с Node.js сървър през Windows Command Prompt и база от данни MySQL през XAMPP.
